# TaylorCommands
Server commands for Fabric 20w15a+

Fork of [Markus23/mycommands](https://github.com/Markus23/mycommands)

## Commands
### Home
#### Delete a home
`/delhome`

#### Warp to a set home
`/home`

#### Set a home
`/sethome`

### Teleport
#### Back to previous location
`/back`

#### Randomly teleport
`/rndtp`

#### Accept a teleport request
`/tpa`
`/tpaccept`

#### Deny a teleport request
`/tpdeny`

### Warp
#### Delete a warp
`/delwarp`

#### Warp to the spawn
`/spawn`

#### Set the spawn warp
`/setspawn`

#### Set a warp
`/setwarp`

#### Warp to a warp point
`/warp`


### Health
#### Restore full health and hunger
`/heal`

### Items
#### Repair item in hand
`/repair`

### Weather/Time
#### Set to daytime
`/day`

#### Set weather to raining
`/rain`

#### Set weather to sunny
`/sun`

### Player
#### Enable flying (like in creative)
`/fly`

#### Make player invulnerable
`/god`
